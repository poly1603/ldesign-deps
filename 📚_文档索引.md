## 🎓 学习路径

### 第一步：了解工具 (15 分钟)

1. 阅读 [START_HERE.md](./START_HERE.md)
2. 运行 `ldeps --help`
3. 尝试 `ldeps check`

### 第二步：基础使用 (30 分钟)

1. 阅读 [快速开始.md](./快速开始.md)
2. 运行 `ldeps interactive`
3. 运行 `ldeps audit`
4. 查看 [examples/basic-usage.ts](./examples/basic-usage.ts)

### 第三步：深入学习 (1-2 小时)

1. 阅读 [docs/CLI_GUIDE.md](./docs/CLI_GUIDE.md)
2. 尝试所有 CLI 命令
3. 配置 `.depsrc.json`
4. 阅读 [docs/BEST_PRACTICES.md](./docs/BEST_PRACTICES.md)

### 第四步：高级应用 (2-4 小时)

1. 阅读 [docs/api.md](./docs/api.md)
2. 查看 [examples/advanced-usage.ts](./examples/advanced-usage.ts)
3. 编写自定义脚本
4. 集成到 CI/CD

### 第五步：成为专家 (持续学习)

1. 阅读所有文档
2. 研究源代码
3. 贡献代码
4. 分享经验
